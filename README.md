# Recipes

### Tech Stack:

- Angular
- NestJS
- Prisma
- MongoDB Atlas

### Features

- JWT Authentication
- CRUD Functionality
- Search
- Likes
- Comments

### Install packages

```shell
npm install
```

### Setup .env file

```js
DATABASE_URL=
JWT_SECRET=
```

### Setup Prisma

```shell
npx prisma db push
```

### Start
```shell
nest start --watch
ng serve
```
